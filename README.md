# মসজিদ আর্থিক ব্যবস্থাপনা অ্যাপ্লিকেশন (মোবাইল-ফার্স্ট ওয়েব অ্যাপ)

এটি একটি মোবাইল-ফার্স্ট সিঙ্গেল পেজ ওয়েব অ্যাপ্লিকেশন (SPA) যা গ্রামের মসজিদের আর্থিক লেনদেন, জুমার চাঁদা, সাধারণ অনুদান এবং সদস্যদের মাসিক চাঁদা ট্র্যাকিং সহজ করার জন্য ডিজাইন করা হয়েছে। এটি মোবাইল ফোনের ব্রাউজারে নিখুঁতভাবে চলবে এবং খুব সহজেই **Vercel**-এ বিনামূল্যে ডেপ্লয় করা যাবে।

## মূল ফিচারসমূহ:
১. **রিয়েল-টাইম ফান্ড ব্যালেন্স:** ক্যাশ ইন হ্যান্ড এবং ব্যাংক অ্যাকাউন্ট ব্যালেন্স লাইভ ট্র্যাকিং।
২. **সদস্যদের প্রোফাইল ও মাসিক চাঁদা ট্র্যাকিং:** ৬০ জন সদস্যের তালিকা (সাধারণ, দরিদ্র ও ফ্রি ক্যাটাগরি) এবং প্রতি মাসের চাঁদা পরিশোধ/বকেয়া ট্র্যাকিং।
৩. **লেনদেন এন্ট্রি:** সহজ ও দ্রুত যেকোনো আয় বা ব্যয়ের তথ্য ইনপুট দেওয়ার ব্যবস্থা।
৪. **মাসিক খতিয়ান ও চার্ট বিশ্লেষণ:** প্রতি মাসের আয় বনাম ব্যয়ের গ্রাফ ও খাতভিত্তিক বিশদ তালিকা।
৫. **প্রিন্ট এবং পিডিএফ তৈরি:** অডিটের সুবিধার্থে এক ক্লিকে সম্পূর্ণ রিপোর্টের প্রিন্ট ও পিডিএফ ডাউনলোড করার সুবিধা।
৬. **লোকাল ডাটা স্টোরেজ ও রিসেট সুবিধা:** ক্যাশিয়ারের ফোনের লোকাল স্টোরেজে তথ্য সংরক্ষিত থাকবে এবং ইন্টারনেট ছাড়াই ব্যবহার করা সম্ভব।

## কিভাবে Vercel-এ ডেপ্লয় করবেন (How to Deploy to Vercel)

অ্যাপটি খুব সহজেই Vercel-এ হোস্ট করতে পারবেন নিচের পদ্ধতিগুলো অনুসরণ করে:

### পদ্ধতি ১: Vercel CLI দিয়ে সরাসরি আপলোড (সবচেয়ে সহজ)
আপনার কম্পিউটারে Node.js ইনস্টল থাকলে নিচের কমান্ড দিয়ে কয়েক সেকেন্ডে ডেপ্লয় করতে পারেন:
1. আপনার কমান্ড প্রম্পট বা টার্মিনাল ওপেন করুন এবং প্রজেক্ট ফোল্ডারে যান:
   ```bash
   cd C:\Users\QC\.gemini\antigravity\scratch\mosque-finance-app
   ```
2. Vercel ইনস্টল না থাকলে ইনস্টল করুন:
   ```bash
   npm install -g vercel
   ```
3. ডেপ্লয় করুন:
   ```bash
   vercel
   ```
   *(এখানে Vercel একাউন্ট লগইন করতে বলবে এবং কয়েকটি প্রশ্নের উত্তর দিলেই ডেপ্লয় সম্পন্ন হবে।)*

### পদ্ধতি ২: GitHub এর মাধ্যমে ডেপ্লয়
1. আপনার GitHub একাউন্টে একটি নতুন রিপোজিটরি (Repository) তৈরি করুন।
2. এই প্রজেক্ট ফোল্ডারের ফাইলগুলো সেখানে আপলোড (Push) করুন।
3. [Vercel Dashboard](https://vercel.com/dashboard)-এ যান।
4. **Add New > Project** এ ক্লিক করুন।
5. আপনার GitHub রিপোজিটরিটি সিলেক্ট করে **Deploy** বাটনে ক্লিক করুন।
6. Vercel কয়েক সেকেন্ডের মধ্যে আপনাকে একটি লাইভ লিঙ্ক (`https://your-project.vercel.app`) দিয়ে দেবে, যা যেকোনো মোবাইল ফোন থেকে সরাসরি ব্যবহার করা যাবে।
